# odin-landing-page
First whole web page design project. Not an easy task, but managed to get it done. Materials and assignments can be seen [here](https://www.theodinproject.com/lessons/foundations-landing-page). This is another project that is not built to be responsive (the design just work on a full screen), but that's something I'm excited to learn because I want to be able to create responsive websites so my projects can be seen on mobile as well as desktop.

## Learning outcomes
 - I've learned how to copy the design from the reference
 - I've learned how to break complex HTML structures into sections
 - I've learned that turning to Google for things that I don't know or can't remember is normal
 - I've learned how to host my projects on the web using GitHub Pages
